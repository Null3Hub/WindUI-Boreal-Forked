# WindUI Boreal

**WindUI Boreal** is a refined fork of the original **WindUI** UI library for Roblox, designed specifically for **Script Hubs** and **Luau environments**.

This fork preserves compatibility with the original API and overall structure while introducing **refinements, bug fixes, and visual improvements**.
